Title : Observability to inverse source problems for linear systems

Abstract

: 

In this talk, we present a link between the problem of observability/controllability and some inverse problems. More precisely,
we show that if a system is exactly observable (i.e. if its adjoint is exactly controllable), then we can recover the source in a corresponding
system if we know the observation. We apply this theory in the example of plate equation: we can recover the position associated to a dirac function
when the source is of the form $\lambda(t)\delta_\xi$.
